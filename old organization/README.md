# IRISTEM Website

Static website for the IRISTEM organization.

## Contents
- `index.html` is the main landing page.
- Other top-level `*.html` files are additional site pages.
- `css/`, `js/`, `img/`, and `fonts/` contain site assets.
- `scss/` contains source styles.

## Viewing Locally
Open `index.html` in a browser.

## Notes
This repo contains vendor assets and a legacy `readme.txt` from the original template.
